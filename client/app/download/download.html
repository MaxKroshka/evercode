Nevercode Sublime Extension

Easily integrate your code snippets and templates with sublime's snippet templates

Download Instructions

Note: this is only supported on Macintosh Operating systems
  1. Download the compressed file below and unzip it in a local directory
  2. Run the setup file by clicking on 'setup.command' If this doesn't run in a terminal emulator, navigate to the folder in your terminal and run './setup.command'.
  3. A prompt will appear asking for your valid Nevercode credentials. Enter them and allow the setup script to run.
  4. Once setup is complete, your Sublime Editor should now contain a syncing option in your 'preferences' menu. Click this option to automatically sync your local snippets with the ones on your account.
  5. All trigger words are prefixed by 'NC_'. So for example, if you choose the trigger word 'express_setup', the actual tab trigger word would be 'NC_express_setup'
