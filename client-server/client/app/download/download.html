Download Instructions

Note: this is only supported on Macintosh Operating systems
  1. Download the compressed file below and unzip it in a local directory
  2. Run the setup file by clicking on 'setup.command' If this doesn't run in a terminal emulator, navigate to the folder in your terminal and run './setup.command'. (Modify the executable permissions by typing 'chmod +x ./setup.command' in your terminal)
  3. A prompt will appear asking for your valid Nevercode client secret. Copy it above using the clipboard button, paste it into the setup script, and allow the script to run.
  4. Once setup is complete, your Sublime Editor should now contain a syncing option in your 'preferences' menu. Click this option to automatically sync your local snippets with the ones on your account.
  5. All trigger words are prefixed by 'NC_'. So for example, if you choose the trigger word 'express_setup', the actual tab trigger word would be 'NC_express_setup'
  6. Should your client-secret ever become compromised, generate a new client secret using the button above and then repeat the above installation in order to reinstall
